# Dynamic PromoCarousel Refactor

This track focuses on making the landing page's promotional carousel truly dynamic by leveraging the `Banner` system.

## Key Files
- `app/(main)/page.tsx`: Contains the `PromoCarousel` component.
- `app/api/admin/banners/route.ts`: API route for fetching banners.
- `components/admin/BannerEditor.tsx`: Reference for the banner configuration structure.
